# Karl Homepage

Next.js version of the Karl homepage.

## Requirements

- Node.js 20 or newer
- npm

## Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Useful Commands

```bash
npm run build
npm run start
npm run lint
```

You can also use the Makefile:

```bash
make install
make dev
make build
```
